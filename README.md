## What is ui-skills? 🛠️

ui-skills acts as a bridge for design engineers who work at the intersection of visual design and technical implementation. This software streamlines your ability to package assets, manage design tokens, and verify interface accuracy before code deployment. Engineers use this tool to ensure the final product matches the initial vision without losing quality during the handoff process.

The application operates as a standalone utility on your Windows machine. It removes the friction of manual export processes and helps you maintain consistency across various projects.

## 🎯 Core Features

### Asset Management
The software exports design assets in formats ready for web and mobile use. It handles scaling, resizing, and compression automatically. You no longer need to crop or export images one by one.

### Design Token Sync
Design tokens represent the building blocks of your interface, such as colors, typography settings, and spacing values. This tool maps these tokens directly to standard variables. This creates a single source of truth for your development team.

### Interface Verification
Compare your live interface against design mockups using the built-in overlay feature. Adjust the opacity to see pixel-perfect alignment. This helps you identify layout drift before you commit changes to the repository.

### Component Auditing
Scan your active project for inconsistent styles. The tool generates a report of elements that deviate from your established design system. You can quickly spot buttons, headers, or forms that require updates.

## ⚙️ How to use the interface

When you open the application, you see a clean dashboard. To start, link your project folder to the software. Click the "Add Project" button and select the root directory of your project. The app analyzes your files and presents a list of assets that require processing.

From the sidebar, select the output format you need. You can toggle between different platforms like CSS, SCSS, or JSON. Click "Execute" to start the processing task. A progress bar shows you the remaining time for the operation.

If you make changes to your design files, the software refreshes the data automatically. You do not need to restart the application.
